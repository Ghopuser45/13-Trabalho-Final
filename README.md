# 13-Trabalho-Final - 📚 Organizador de Estudos

## Nome(s): Gustavo Henrique, Kauan Pereira, Ian Maia, Isly do Nascimento

## Sobre o trabalho

O Organizador de Estudos é uma aplicação web desenvolvida para auxiliar estudantes no gerenciamento de atividades acadêmicas, permitindo adicionar e acompanhar a organização de tarefas de forma simples e intuitiva.

O sistema foi desenvolvido utilizando tecnologias de Front-End e tem como objetivo facilitar o planejamento dos estudos e o acompanhamento de prazos de entrega.

## Objetivos do organizador de estudos

- Organizar atividades acadêmicas.
- Acompanhar o progresso das tarefas.
- Identificar prazos próximos de vencimento.
- Melhorar a produtividade dos estudantes.
- Oferecer uma interface simples e fácil de utilizar.

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
- LocalStorage

## Funcionalidades

- Cadastro de tarefas.
- Edição de tarefas.
- Exclusão de tarefas.
- Descrição opcional das atividades.
- Definição de prioridade (Alta, Média ou Baixa).
- Controle de status (Pendente, Em andamento e Concluída).
- Dashboard simples com estatísticas.
- Barra de progresso.
- Filtros de visualização.
- Avisos de prazo.
- Modo escuro.
- Armazenamento local dos dados.
